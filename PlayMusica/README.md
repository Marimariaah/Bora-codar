# 🎵 Play Music

Um reprodutor de música interativo construído com HTML, CSS e JavaScript. Projeto desenvolvido como parte do desafio **Bora Codar** da Rocketseat.

## 📋 Sobre o Projeto

Play Music é uma interface de reprodutor de áudio responsiva e moderna que apresenta dois layouts diferentes de players: um layout vertical compacto e outro layout horizontal expandido. O projeto demonstra habilidades em estrutura HTML semântica, estilização CSS avançada e diagramação com Grid.

## ✨ Funcionalidades

- 🎼 **Dois Layouts de Player**: Versão vertical (compacta) e horizontal (expandida)
- ⏯️ **Controles de Reprodução**: Play, próxima música e música anterior
- 🖼️ **Visualização de Capa**: Imagens da capa do álbum/música
- 📊 **Barra de Progresso**: Indicador visual do tempo de reprodução
- 📱 **Design Responsivo**: Interface adaptável para diferentes tamanhos de tela
- 🎨 **Tema Escuro**: Interface moderna com paleta de cores escura

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica
- **CSS3**: Layout com Grid, Flexbox e animações
- **Google Fonts**: Fonte Roboto para tipografia
- **Material Icons**: Ícones SVG para controles

## 🚀 Como Usar

1. **Clone ou baixe o repositório**
   ```bash
   git clone https://github.com/Marimariaah/Play-music.git
   cd Play-music
   ```

2. **Abra o arquivo `index.html` no navegador**
   - Clique duas vezes no arquivo `index.html`
   - Ou clique com o botão direito e selecione "Abrir com" > seu navegador preferido

## 📝 Notas sobre o Código

- Os players utilizam a técnica de **CSS Grid** para posicionamento
- As imagens são carregadas do Unsplash (recursos externos)
- Os ícones são SVG inline para máxima compatibilidade
- O design foi pensado para demonstrar habilidades de UI/UX
